# File Tools

This category contains utilities for file management and organization.

## Subfolders

- `filename-generators/` — automation scripts for generating file names and logs.
- `filenames/` — filename normalization, capitalization, and cleanup utilities.
- `indexing/` — directory indexing and archive scanning tools.
- `merger/` — directory merge tools with conflict handling.
- `mover/` — file move and transfer utilities.
- `sorter/` — file sorting utilities.
- `tags/` — metadata tagging tools for media files.
- `zipper/` — compression and archive-related scripts.

## Recommended usage

Use `banshee.rb` for unified execution where possible, or run scripts directly inside these folders when you need specialized behavior.
