# Zipper Tools

This folder contains archive and snapshot utilities for script backups.

## Active tool

- `zscr9.rb` — the current fast snapshot/archive tool.
  - creates `tar.gz` snapshots of selected directories
  - spreads copies to configured destination drives
  - supports Windows, WSL, and Linux
  - includes a fast progress UI and dry-run support

## Legacy versions

Older zipper scripts are stored in `tools/file/zipper/legacy/`:
- `zscr3.rb`
- `zscr5.rb`
- `zscr6.rb`
- `zscr7.rb`
- `zscr8.rb`
- `zscr_1.rb`

Those legacy scripts are kept for reference only and are not part of the main active toolset.
