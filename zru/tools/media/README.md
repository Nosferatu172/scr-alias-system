# Media Tools

This category contains utilities for media processing and conversion.

## Subfolders

- `jpg2pdf/` — convert images to PDF.
- `meta/` — metadata removal and media cleanup tools.
- `music/` — music-related utilities and playlists.
- `repair/` — media repair scripts for JPG, MP3, MP4 files.
- `yt/` — YouTube downloaders and yt-dlp related scripts.

## Recommended usage

Use `banshee.rb` for cross-cutting media actions such as conversion, metadata cleaning, or YouTube downloads. Scripts in this folder are useful for standalone media workflows.
