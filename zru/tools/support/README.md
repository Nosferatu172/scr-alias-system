# Support Tools

This category contains supporting utilities, logging helpers, profiles, and user interface scripts.

## Subfolders

- `finrep/` — financial reporting utilities.
- `logger/` — logger appender and logging framework scripts.
- `profile/` — profile and environment helper scripts.
- `rel0/` — release and version-related helper scripts.
- `UI/` — user interface scripts and interactive helpers.

## Recommended usage

Use these scripts to support your main workflows, especially when you need logging, profile data, or UI enhancements. The main `banshee.rb` tool does not replace all of these support utilities, but it provides a unified entry for general file and media work.
