# ZRU Tools Collection

This folder contains categorized tool collections for file, media, text, password, support, and testing utilities.

## Categories

- `tools/file/` — file system utilities, filename management, merge and archive helpers.
- `tools/media/` — media workflows, image/PDF conversion, metadata cleanup, YouTube helpers, and repair tools.
- `tools/text/` — text processing, JSON utilities, and text manager scripts.
- `tools/password/` — password generation utilities.
- `tools/support/` — support utilities including logging, profiles, UI helpers, and financial reporting.
- `tools/testing/` — development and environment test scripts.

## Active tools

- `banshee.rb` — the unified command-line toolbox for most workflows.
- `tools/file/zipper/zscr9.rb` — the active snapshot/archive tool for fast script backups and restoration.

## Legacy vs active

- Active tools live directly in the category folders.
- Older versions and replaced scripts are kept in `tools/.../legacy/` or `deprecated/` for reference.

## Recommended workflow

1. Use `banshee.rb` for general file and media tasks.
2. Use `tools/file/zipper/zscr9.rb` for snapshot archiving and fast restores.
3. Review legacy tools in `deprecated/` or `tools/.../legacy/` only when you need historical behavior.
